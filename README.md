# 🕌 Manevi Rota

**EN** | [TR](#türkçe)

An AI-powered web application that generates personalized one-day spiritual and nature routes across Turkey.

## Features
- Select from 20+ Turkish cities
- Choose route type: Spiritual, Nature, or Both
- AI-generated personalized daily itineraries
- Save and manage your favorite routes
- Clean, responsive design

## Technologies
- HTML5, CSS3, JavaScript (ES6)
- OpenRouter API (google/gemma-3-4b-it:free)
- LocalStorage for data persistence

## Setup
1. Clone the repository
```bash
   git clone https://github.com/HavvaNur-A/manevi-rota.git
```
2. Create a `config.js` file in the root folder:
```javascript
   const API_KEY = "your-openrouter-api-key";
```
3. Open `index.html` in your browser

## Get API Key
Free API key available at [openrouter.ai](https://openrouter.ai)
